# jupyter_counter

Jupyter代码行数统计工具

![image](https://github.com/Nonumx/jupyter_counter/assets/17672640/f66655d6-9577-4fd9-895f-1bc7cf4833cd)

## Getting Started

克隆并编译本仓库

```powershell
git clone https://github.com/Nonumx/jupyter_counter.git
cd jupyter_counter
flutter build windows # 或者你想编译linux，mac，都行
```
